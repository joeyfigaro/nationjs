# Command line applications in Node.js (josh finnie)

## Prepare your CLA
1. *set global preference in package.json*
`// package prefers to be installed globally
preferGlobal: true
`
2. *provide location of executable with bin*
3. *add credentials for npm*
`npm adduser`
4. *shrinkwrap deps*
5. *publish to NPM*
`npm publish`
